# 🏴‍☠️ One Piece Discord Bot

Un bot Discord RPG sur le thème de One Piece avec fruits du démon, combats PvE et PvP !


## Commandes

| Commande | Description |
|----------|-------------|
| `!start` | Créer ton personnage |
| `!profil [@joueur]` | Voir les stats |
| `!gacha` | Obtenir un Fruit du Démon (200 Berry, 1h CD) |
| `!combat` | Affronter un monstre (30s CD) |
| `!pvp @joueur` | Duel PvP contre un autre joueur |
| `!soin` | Récupérer 50% des HP (10 min CD) |
| `!fruits` | Liste de tous les Fruits du Démon |
| `!classement` | Top 10 des pirates |
